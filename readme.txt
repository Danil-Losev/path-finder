Комментарии:
    
    Все изображения являются частью exe файла. Так что если надо изменить или добавить
    новое изображение то соответствующие пути нужно добавить в файл ресурсов и массив 
    подгружаемых изображений. Так сделанно специально для удобства сборки 
    и переноса приложения

    Проект залит на GitHub https://github.com/Danil-Losev/path-finder
    В раздел Releases добавлю архив с билдом, так как его размер в этот раз большой

    Компилятор: MSVC 2022

    Важно!!! Проект реализован с использованием фреимворка QT6. 
        Если он у Вас не установлен. То разархивируйте архив build. 
        Там уже есть все не необходимые dll

    Программа реализована послойно:
        Есть слой движка и слой UI

    В движке на уровне кода реализована возможность использования
    разных фильтров. Есть генератор шума. 
    Во время бинарной обработки изображения. для слишком зашумлённых 
    изображений, реализована возможность искусственного удаления шума 
    по определённой площади

    Объяснения и рекомендации:
    1) Реализован графический интерфейс: для этого можно просто запустить приложение

    2) Сборка проекта возможна как через Visual Studio (открыть представление папки, и сгенерировать CMakeCache.txt)
        так и в ручную через команды (перед этим измените в CMakeLists.txt путь к windeployqt.exe)
            * cmake -S . -B build -DCMAKE_PREFIX_PATH="путь к qt для компилятора msvc2022"
            * cmake --build build --config Release или cmake --build build --config Debug 
            * Вызвать windeployqt.exe для полученного exe файла если не появились dll
